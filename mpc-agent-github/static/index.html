MPC Auto Config Assistant

MPC 自动参数配置助手

现在支持“自然语言 + 结构化选项”混合输入。即使用户说不清完整需求，也可以先从参与方规模、电路类型、敌手模型、门限和安全目标这些维度逐步收敛。

1. 填写自由需求或点选 MPC 属性 2. 生成结构化需求与协议候选 3. 可选执行 MP-SPDZ 或 SecretFlow SPU
